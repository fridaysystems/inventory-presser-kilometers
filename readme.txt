Uses kilometers as the units for vehicle odometer values in themes designed for Inventory Presser

== Description ==

An add-on for the Inventory Presser dealership plugin that changes odometer units to kilometers.

== Installation ==

The quickest method for installing this plugin is:

1. Upload the `kilometers-instead-of-miles` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
